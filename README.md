# h5-huangguan-archive

## 项目简介

`h5-huangguan-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的集合，方便版本管理与历史追溯。

## 目录说明

```
h5-huangguan-archive/
├── index.html          # 入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
├── assets/             # 公共资源文件（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/`：按日期或功能命名的子目录，每个子目录内放置一个完整的 HTML 页面及其依赖资源。
- `assets/`：多个页面共享的样式、脚本或图片，避免重复存放。

## 页面归档说明

本仓库的每个 HTML 页面均为独立归档单元，不依赖外部服务或数据库。归档遵循以下原则：

- 每个页面包含完整的 HTML 结构与必要的样式、脚本。
- 页面之间无耦合，可单独打开使用。
- 归档版本通过 Git 提交记录进行追踪，便于回溯与比较。

## 维护说明
